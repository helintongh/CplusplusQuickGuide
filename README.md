运算符重载有什么用?要明白运算符重载与泛型算法密切相关。
1. 泛型算法参数接收的都是迭代器,泛型算法往往是给所有容器用的全局函数,泛型算法有一套方法能够统一的遍历所有的容器的元素(通过迭代器)。迭代器一般实现为容器的内部嵌套类型,因为不同容器操作不同。
2. 而迭代器大量运用到了运算符重载。
